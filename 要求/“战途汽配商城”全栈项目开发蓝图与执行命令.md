# “战途汽配商城”全栈项目开发蓝图与执行命令

> **目标**: 一次性生成可直接部署运行的企业级汽配电商平台，包含后端服务、Web管理后台、安卓用户端。
> **基准规范**: 严格遵循《企业级实际开发的流程.md》中的代码风格、分层结构和异常处理规范。

## 一、项目总体要求

1. **技术栈**:
   - **后端**: Spring Boot 3.2.x + MyBatis-Plus 3.5.x + MySQL 8.0 + Redis (Jedis/Lettuce) + JWT + Swagger 3 (springdoc-openapi)
   - **Web前端**: Vue 3 + Vite + Element Plus + Pinia + Axios
   - **安卓前端**: Kotlin + Jetpack Compose + Retrofit + Coil (图片加载)

2. **开发原则**:
   - 后端严格按照企业级流程分层: `controller` -> `service` -> `mapper`。
   - 实体类使用 Lombok，MyBatis-Plus 代码生成器可选但需保证字段映射正确。
   - 使用 `Result<T>` 统一响应，`GlobalExceptionHandler` 全局异常处理。
   - **数据完整性**: 必须将我提供的所有商品数据（见附件描述）通过 SQL 脚本或启动初始化器导入数据库，确保启动后即有数千条真实商品数据。
